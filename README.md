# turmite-rule-simulator
A small University project that allows users to define custom Turmite rules and run them. Read more on turmites: https://en.wikipedia.org/wiki/Turmite

## Run it yourself!

Find the latest JAR file in the Releases section, and download it. You can double-click to run it, or run:

```bash
java -jar turmite-rule-simulator-1.0.jar
```

## Or build it!

You can clone the repo and build it yourself. After this, you can run it the same way as described above.

Build the JAR with the following command:

```bash
.\mvnw clean compile assembly:single
```
